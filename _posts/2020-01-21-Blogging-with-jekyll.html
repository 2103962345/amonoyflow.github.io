What is Jekyll?

Jekyll is a Ruby Gem and it helps you generate a static website ('static website generator'). Jekyll takes it's content from Markdown, Liquid or HTML and comes out as a static page through your template ready to be deployed on GitHub.

Quickstart

1. Install Ruby development environment.

2. Open Powershell or commandline then install Jekyll and budler gems.

gem install jekyll bundler

3. Create jekyll site.

jekyll new myWebsite

4. Go to newly created website directory.

cd myWebsite

5. Build the site.

bundle exec jekyll serve

6. Browse http://localhost:4000

Jekyll themes

Jekyll community also provides themes that you can use.

- jekyllthemes.io

- jamstackthemes.dev

- jekyll-themes.com

Host on GitHub

1. Upload your code on your GitHub account.

2. Go to settings and change the repository name to <name>.github.io

3. Under 'GitHub Pages' change source to master.
